# 2026-04-25 시험 — SDK Media Settings G1~G4 facade

## 대상

- 코드 변경: `core/pipe.js`, `core/device-manager.js`, `core/engine.js`, `core/filters/background-blur-filter.js`(신규), `core/filters/noise-cancel-filter.js`(신규)
- 설계서: `context/design/20260425_sdk_media_settings.md`
- 시험 대상 API:
  - G1: `engine.setVideoBitrate(bps)`, `engine.setVideoFps(fps)`, `engine.setVideoSize(w,h)`, `pipe.degradation`, `pipe.getBitrate()`
  - G2: `engine.outputMuted` (set/get), `engine.outputVolume` (set/get)
  - G3: `engine.setMicGain(value)`, `engine.micGain` (get), `engine.setMicCapture(opts)`, `engine.micCapture` (get)
  - G4: `engine.setBackgroundBlur(strength|null)`, `engine.setNoiseCancel(enabled)`

## 판정 기준

- **성공**: 호출이 완료되었고, 반환값과 측정값이 본 시험에서 정의한 expected 와 일치
- **실패**: 호출 예외 발생 또는 반환값/측정값이 expected 와 불일치
- **알수없음**: 본 회차 환경에서 expected 비교에 필요한 측정값을 얻을 수 없음 (다음 회차에서 재시험 필요)
